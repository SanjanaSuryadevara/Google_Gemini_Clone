# 🤖 Google Gemini Clone

A modern AI chatbot web application inspired by **Google Gemini**, built using **HTML, CSS, and JavaScript**. The project provides an intuitive conversational interface where users can interact with an AI assistant through a clean, responsive, and user-friendly design.

## 🚀 Features

- 💬 Interactive AI chat interface
- 🎨 Modern and responsive UI
- 🌙 Clean Gemini-inspired design
- ⚡ Real-time AI responses
- 📱 Fully responsive across devices
- 🔄 Auto-scrolling chat window
- 👤 User and AI message bubbles
- 📋 Copy AI responses
- ⌨️ Send messages using Enter key
- ⏳ Typing/loading animation
- 🛡️ Basic input validation

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Google Gemini API *(or your configured AI API)*

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/google-gemini-clone.git
```

### Navigate to the project

```bash
cd google-gemini-clone
```

### Open the project

Simply open **index.html** in your browser.

or use VS Code Live Server.

---

## 🔑 API Configuration

If your project uses the Google Gemini API:

1. Create an API key from Google AI Studio.
2. Open `script.js`
3. Replace the placeholder key with your own.

```javascript
const API_KEY = "YOUR_API_KEY";
```

> Never commit your real API key to GitHub.
